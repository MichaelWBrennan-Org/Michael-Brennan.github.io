Accessibility Statement

Infinite Match - Match-3 Puzzle Game

Last Updated: October 2025

Michael Brennan ("we," "our," or "us") is committed to ensuring that Infinite Match is accessible to players of all abilities. We believe that gaming should be inclusive and enjoyable for everyone.

WCAG 2.1 AA Compliant
Conformance Level: We strive to meet the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards for game interfaces and web-based components.

Our Commitment

We are committed to making Infinite Match accessible to players with disabilities by:

  • Following WCAG 2.1 AA guidelines for all game interfaces
  • Implementing inclusive design principles from the ground up
  • Regular accessibility testing and user feedback integration
  • Continuous improvement based on accessibility best practices
  • Providing alternative access methods when needed

Visual Accessibility

We have implemented several features to support players with visual impairments:

Color and Contrast

  • High Contrast Mode: Alternative color scheme with enhanced contrast ratios
  • Color Blind Support: Game elements distinguishable by shape and pattern, not just color
  • Customizable Colors: Players can adjust gem colors to their preferences
  • Visual Indicators: Clear visual feedback for all game actions

Text and UI Elements

  • Scalable Text: All text can be enlarged through device settings
  • Clear Icons: Intuitive icons with text labels where appropriate
  • Consistent Layout: Predictable interface layout reduces cognitive load
  • Focus Indicators: Clear focus indicators for keyboard navigation

Motor Accessibility

Infinite Match supports players with motor impairments through:

Input Flexibility

  • Large Touch Areas: Generous touch targets for easier interaction
  • Drag and Drop: Intuitive drag-and-drop mechanics
  • Alternative Controls: Multiple ways to perform the same action
  • Haptic Feedback: Tactile feedback for touch interactions

Timing and Pacing

  • No Time Pressure: Players can take their time with moves
  • Pause Functionality: Game can be paused at any time
  • Undo Options: Ability to undo moves if needed
  • Adjustable Speed: Game speed can be adjusted for comfort

Auditory Accessibility

For players with hearing impairments, we provide:

Visual Alternatives

  • Visual Cues: All audio cues have visual equivalents
  • Text Notifications: Important information displayed as text
  • Vibration Alerts: Haptic feedback for important events
  • Subtitles: Text-based information for all audio content

Audio Customization

  • Volume Controls: Separate volume controls for music, effects, and voice
  • Audio Descriptions: Detailed audio descriptions for visual elements
  • Silent Mode: Game fully playable without audio
  • Custom Sound Effects: Players can customize or disable sound effects

Cognitive Accessibility

Infinite Match supports players with cognitive differences through:

Clear Design

  • Simple Interface: Clean, uncluttered game interface
  • Consistent Patterns: Predictable game mechanics and controls
  • Clear Instructions: Easy-to-understand game rules and objectives
  • Progress Indicators: Clear visual feedback on game progress

Customization Options

  • Difficulty Settings: Adjustable difficulty levels
  • Hint System: Optional hints for challenging levels
  • Save Anytime: Game progress can be saved at any point
  • Restart Options: Easy restart functionality for levels

Assistive Technology Support

Infinite Match is designed to work with various assistive technologies:

Screen Readers

  • VoiceOver (iOS): Full support for Apple's screen reader
  • TalkBack (Android): Compatible with Google's screen reader
  • NVDA/JAWS (Windows): Support for desktop screen readers
  • Voice Control: Voice commands for game control

Switch Navigation

  • Switch Access: Compatible with switch navigation devices
  • Eye Tracking: Support for eye-tracking input devices
  • Head Tracking: Alternative input methods for motor impairments
  • Voice Control: Voice-activated game controls

Testing and Evaluation

We regularly test Infinite Match for accessibility compliance:

Testing Methods

  • Automated Testing: Regular automated accessibility scans
  • Manual Testing: Comprehensive manual accessibility testing
  • User Testing: Testing with players who use assistive technologies
  • Expert Review: Accessibility expert evaluation and recommendations

Testing Tools

  • WAVE: Web accessibility evaluation tool
  • axe-core: Automated accessibility testing
  • Lighthouse: Accessibility auditing
  • Manual Testing: Testing with actual assistive technologies
